# starter react
A ready-to-use starter for a front application based on React and Redux with Webpack

### Run :
1. run `npm install`
2. run `npm start`
3. Go to `http://localhost:3000/` ! :)

### Build the app for prod
1. run `npm build`
2. just put your /build folder online !
